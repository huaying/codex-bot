# Seeking Alpha Research

Created: 2026-05-02

This folder records the current crypto alpha research plan and working theses.

## Research Frame

We treat alpha as a mechanism-level mismatch, not as a public narrative.

```text
Alpha =
  new flow / new demand
+ forced counterparty
+ underpriced bottleneck
- crowded public narrative premium
```

The main 2026 crypto environment assumptions:

- Stablecoins are now core settlement infrastructure.
- DEX perpetuals have become one of the largest onchain fee engines.
- RWA perps are no longer just a narrative; they are active trading venues.
- Cheap blockspace commoditizes infra and shifts value toward apps, orderflow, routing, liquidity, and oracle/control points.
- Regulation is moving from pure uncertainty into a design constraint, especially for stablecoins and RWA products.

## Current Priority

The first active research track is:

```text
RWA commodity perps carry mismatch
```

Why this track:

- It is mechanism-driven.
- It has public data surfaces.
- It is not only directional trading.
- It may create delta-neutral carry opportunities.
- It involves different venue designs that can diverge.
